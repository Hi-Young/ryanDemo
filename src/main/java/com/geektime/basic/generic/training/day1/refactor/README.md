# 🎯 Day 1 重构练习：用泛型消除重复代码

## 练习目标
通过重构一段充满重复代码的程序，**亲身体验泛型的价值**。

## 场景描述

你接手了一个电商系统的代码，发现有3个数据容器类：
- `UserBox` - 存储用户数据
- `ProductBox` - 存储商品数据
- `OrderBox` - 存储订单数据

这些类的代码**90%都是重复的**，只有类型不同！

## 你的任务

### 第1步：阅读重复代码（理解痛点）
打开 `before/` 目录，看看没有泛型时的代码有多痛苦：
- `UserBox.java` - 100行
- `ProductBox.java` - 100行
- `OrderBox.java` - 100行
- **总计：300行重复代码！**

### 第2步：用泛型重构（消除重复）
在 `after/` 目录创建：
- `GenericBox.java` - 用泛型实现通用容器
- **目标：用50行代码替代300行！**

### 第3步：对比差异（理解价值）
运行 `ComparisonDemo.java`，看看重构前后的差异。

## 评估标准

- ✅ 能否用一个泛型类替代3个重复的类？
- ✅ 能否保持原有功能不变？
- ✅ 代码量是否大幅减少？
- ✅ 类型安全是否得到保证？

## 开始练习

```
1. 先看 before/UserBox.java（重复代码的痛苦）
2. 再看 before/ProductBox.java（同样的痛苦）
3. 然后在 after/ 目录创建 GenericBox<T>
4. 用泛型重构，消除重复！
```

**这才是学习泛型的正确姿势！** 🚀
