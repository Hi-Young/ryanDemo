# 🎯 策略模式互动练习

## 📚 学习目标
通过动手实践，深刻理解策略模式的设计思想和实现方法。

## 🚀 练习说明

### 练习1：实现组合促销策略
**任务**：创建一个能够组合多种促销的策略 `ComboPromotionStrategy`

**要求**：
1. 继承 `PromotionStrategy` 接口
2. 能够同时应用多种促销规则
3. 按照最优化原则组合计算
4. 添加 `@Component` 注解让Spring管理

**提示**：
- 注入所有已有的策略
- 遍历计算每种策略的结果
- 选择最优的组合方式
- 生成详细的促销说明

### 练习2：实现限时抢购策略  
**任务**：创建限时抢购策略 `FlashSaleStrategy`

**要求**：
1. 只在特定时间段生效
2. 支持商品分类折扣配置
3. 限制每个用户的购买次数
4. 实现库存检查逻辑

**提示**：
- 检查当前时间是否在活动期间
- 根据商品分类应用不同折扣
- 查询用户历史购买记录
- 验证库存充足性

### 练习3：数据库驱动的动态策略
**任务**：创建从数据库读取配置的动态策略 `DatabaseDrivenStrategy`

**要求**：
1. 从 `promotion_rule_config` 表读取规则
2. 动态解析配置参数
3. 支持规则热更新
4. 处理配置异常情况

**提示**：
- 使用 MyBatis 查询配置
- 缓存配置提高性能
- 解析不同类型的规则值
- 异常时使用默认规则
